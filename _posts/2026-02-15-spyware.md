# Non-spyware software as cited by [spyware.neocities.org](https://spyware.neocities.org/):
<br>  
## Web Browsers
[[link](https://hacktivis.me/projects/badwolf)] [BadWolf](https://spyware.neocities.org/articles/badwolf) (minimalist but usable)<br>
[[link](https://icecatbrowser.org/index.html)] [Gnu IceCat](https://spyware.neocities.org/articles/icecat)<br>
[[link](https://lynx.browser.org/)] [Lynx](https://spyware.neocities.org/articles/lynx) (text-only)<br>
[[link](https://iridiumbrowser.de/)] [Iridium](https://spyware.neocities.org/articles/iridium) (after some [steps](https://spyware.neocities.org/guides/iridium))<br>
[[link](https://www.falkon.org/)] [Falkon](https://spyware.neocities.org/articles/falkon) (DYOR, could have spyware on windows, not on linux)<br>
[[link](https://www.netsurf-browser.org/)] [Netsurf](https://spyware.neocities.org/articles/netsurf) (after some [steps](https://spyware.neocities.org/guides/netsurf.html), extremely lightweight, uses own engine so mostly unsupported, no JS as of 3.11)<br>
[[link](https://otter-browser.org/)] [Otter Browser](https://spyware.neocities.org/articles/otter) (no spyware, but it uses old libraries, most likely with known exploitable vulnerabilities)<br>
[[link](https://www.palemoon.org/)] [Pale Moon](https://spyware.neocities.org/articles/palemoon) (after some [steps](https://spyware.neocities.org/guides/palemoon))<br>
[[link](https://www.qutebrowser.org/)] [Qutebrowser](https://spyware.neocities.org/articles/qutebrowser) (keyboard-focused mostly like vim, minimal GUI)<br>
[[link](https://www.seamonkey-project.org/)] [SeaMonkey](https://spyware.neocities.org/articles/seamonkey) (after some [steps](https://spyware.neocities.org/guides/seamonkey), basic)
[[link](https://surf.suckless.org/)] [Surf](https://spyware.neocities.org/articles/surf) (very basic)<br>
[[link](https://www.torproject.org/)] [Tor](https://spyware.neocities.org/articles/tbb) (after some [steps](https://spyware.neocities.org/guides/tbb), might still have some spyware, changing settings may increase fingerprinting)<br>
[[link](https://github.com/ungoogled-software/ungoogled-chromium)] [Ungoogled-Chromium](https://spyware.neocities.org/articles/ungoogled_chromium) (one of the best Chrome-based spyware-free browser)<br>
[[link](https://git.nuegia.net/webbrowser.git/)] [Web Browser](https://spyware.neocities.org/articles/webbrowser) (based on Pale Moon)<br>
<br>  
## Messaging Clients/Services
[[link](https://hexchat.github.io/index.html)] [Hexchat](https://spyware.neocities.org/articles/hexchat) (standard IRC channels aren't encrypted, might leak IP addresses, local conversations, pretty much only good for privacy, not anonymity, FOSS)<br>
[[link](https://signal.org/)] Signal (not listed, but safer than Hexchat)<br>
[[link](https://qtox.github.io/)] qTox (not listed, public key identification, full anonymity, P2P, IP might be exposed if no VPN, FOSS)<br>
[[link](https://element.io/)] Matrix (not listed, decentralized, can be self hosted, server can know who is talking to whom and message lengths, some unencrypted metadata, user's IP address is exposed to server, FOSS)<br>
(_btw [Discord](https://spyware.neocities.org/articles/discord), kinda funny_)<br>
(_btw [Telegram](https://spyware.neocities.org/articles/telegram) is not perfect_)<br>
<br>  
## Search Engines
[[link](https://duckduckgo.com/)] [DuckDuckGo](https://spyware.neocities.org/articles/duckduckgo) (marked as possibly spyware, but it's the only non-extremely high listed)<br>
[[link](https://www.startpage.com/)] Startpage (not listed, but afaik superior than DuckDuckGo in privacy, while still getting results from Google. DYOR)<br>
<br>  
## Music Players
[[link](https://www.videolan.org/)] [VLC](https://spyware.neocities.org/articles/vlc) (duh)<br>
[[link](https://krosbits.in/musicolet/)] Musicolet (not listed, allegedly no internet connection, downloadable from high-risk sources for privacy, possibly biased so DYOR (i use it))<br>
[[link](https://www.strawberrymusicplayer.org/#about)] Strawberry Music Player (not listed, FOSS, GPL license, Clementine fork, DYOR)<br>
